Monitoring Dashboard for WSO2 Micro Integrator
======================================================================

Welcome to Monitoring dashboard @product.version@ for WSO2 Micro integrator.
This is a light weight UI server hosting the react application that implements the monitoring dashboard.

Running the Monitoring Dashboard
======================================================================
1. Go to the bin directory and run the dashboard.sh file.
2. Load the login page with the dashboard context. i.e: https://localhost:9743/dashboard/login
